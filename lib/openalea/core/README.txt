
Documentation
-------------

  * Overview of the project contained in ./doc/user/overview.txt
  * Full documentation including API, User Guide and tutorial can be found in
    the  ./doc directory. In order to compile the HTML documentation, type::

    python setup.py build_sphinx.

    Sphinx, version 0.6 is required.
  * In a python interpreter:: 
        
        import openalea.core
        help(openalea.core)
  * Openlea Main URL: http://openalea.gforge.inria.fr
  * Online documentation: http://openalea.gforge.inria.fr/doc/sphinx/core
