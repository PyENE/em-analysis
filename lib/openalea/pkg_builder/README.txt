===== About =====

=== Description ===

Package Builder is a package to build other OpenAlea packages.
It generates, based on OpenAlea guidelines, a layout structure with templates files when possible.

It can be used directly from python or from a shell.
It will be integrated in OpenAlea and in Visualea.
